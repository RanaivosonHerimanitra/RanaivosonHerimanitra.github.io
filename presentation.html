From Rmarkdown to slidify

October 31, 2014 by Herimanitra

This is not a tutorial by itself but rather a guide of the top framework to work on data and present as well. If you've just came into the R world and just discovered RStudio then you probably may curious about Rmarkdown.

Rmarkdown is an extended markdown within R that allows us to write and execute r-code behind the scene as well as the text (such as equation) that describes the data processing.

Rmarkdown comes to be very useful when one needs to quickly present their work to a broad audience. For example, here is an Rmarkdown I wrote during a lab to introduce students to R and linear programming:

Shiny is an R package for statistical web application but not only. You can present a technical work in a way that attract the audience attention. You will see later that one can insert a shiny application into a slidify framework.
